<script type="text/javascript">
$(function (){
	$(".manhuaTip").manhuaTip({					       
		Event : "click",//可选				       
		timeOut : 200//停留时间
	});
});
</script>
用法如下：
1、一共有4种提示操作，1、错误（error）  2、 成功（succ）  3、消息（notice）  4、等待（loading）
2、每个使用的元素加上以下3种属性
   1）class必需为manhuaTip
   2) ty类别必需为第一条的4种之一
   3）msg提示的消息
用法如下：
<input type="button" value="错误" class="manhuaTip" ty="error" msg="出错了" />
<input type="button"  value="成功" class="manhuaTip" ty="succ" msg="提示插件发布成功了" />
<input type="button"  value="消息" class="manhuaTip" ty="notice" msg="提示插件发布了" />
<input type="button"  value="等待" class="manhuaTip" ty="loading" msg="正在提交您的请求，请稍候..." />
